# EVREKA_Application
Evreka Back-End Developer Evaluation Responses

This project enters data about vehicle plate number, location and time to the NavigationRecord model using the http://127.0.0.1:8000/insert_data url. This information is stored in an SQLite3 database. 
Information about the last location entry sent by each vehicle may be retrieved using the http://127.0.0.1:8000/read_data url. Django ORM framework is used for queries.
